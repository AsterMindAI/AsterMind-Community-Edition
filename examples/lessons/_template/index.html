Welcome to your lesson

This is the AsterMind lesson template. Copy this directory to examples/lessons/L<NN>-<slug>/, edit the five files, and you have a new lesson.

The five files in every lesson:

  • index.html — the deck shell (this file)
  • slides.json — slide list as data
  • live-demo.js — interactive demos wired to slides
  • speaker-notes.js — what to emphasize per slide
  • README.md — pitch, prereqs, learning outcomes, you-try

Use the Next button or arrow keys to advance.

How to add a slide

Three places to edit, in any order:

// 1. index.html — add a section
<section class="slide" id="my-new-slide">
  <h2>My new slide</h2>
</section>

// 2. slides.json — add the slide to the order
{ "id": "my-new-slide", "title": "My new slide" }

// 3. speaker-notes.js — add notes (optional)
"my-new-slide": { left: "...", right: "..." }

If you forget to add the slide to slides.json, the deck falls back to DOM order. If you forget speaker notes, the slide just won't have any. Nothing crashes.
