Summer 2026 — Internship Curriculum

Browser-first machine learning,
taught hands-on, end-to-end.

Seven interactive lessons that take you from "what's a neural net?" to shipping a real ML demo in production code. Built around AsterMind's Extreme Learning Machine library — train models in milliseconds, in a tab, no GPU. Each lesson ends with a hands-on you-try; the final five weeks are a per-intern capstone you'll demo on-site July 24.

Foundation lessons

L00 → L06 — work through them in order; each builds on the last.
L00 · 45 min

ELM Primer

What an Extreme Learning Machine is and why solving in closed form (not backprop) is a clever idea. Slide-deck walk-through with a live demo.

45 min ✓ shipped
L01 · 45 min

JS / TS in this repo

Clone, install, run 106+ tests green, ship a one-line PR. The dev-loop on-ramp — no ML, just fluency in the tools you'll use every day.

45 min ✓ shipped
L02 · 45 min

Your first classifier

Train an IntentClassifier on a 20-example greeting set. Hit ≥80% on a held-out test. Add a third class. Export the snapshot. Live in the page.

45 min ✓ shipped
L03 · 45 min

Embeddings + similarity

Encode text into vectors. Query an EmbeddingStore by cosine similarity. Find the failure mode where lexical and semantic similarity diverge.

45 min ✓ shipped
L04 · 45 min

Online learning

Stream data into OnlineELM one sample at a time. Watch accuracy climb. Trigger concept drift mid-stream and tune the forgetting factor to recover.

45 min Thomas req.
✓ shipped
L05 · 45 min

Classification with confidence

Threshold tuning + live precision/recall + confusion matrix on a binary support-ticket dataset. Find the threshold that meets a real operational bar.

45 min Jarrett req.
✓ shipped
L06 · 60 min

Kernels and Nyström

When linear models fail, the kernel trick succeeds. Train RBF, sweep Nyström landmarks. Speed/accuracy tradeoff with a live decision-boundary visualization.

60 min Nolan req.
