## Migration from Old Packages

If you were using any of the old packages, simply:

1. Uninstall old package:
   ```bash
   npm uninstall @astermind/astermind-elm
   # or
   npm uninstall @astermind/astermind-pro
   # or
   npm uninstall @astermind/astermind-premium
   # or
   npm uninstall @astermind/astermind-synthetic-data
   ```

2. Install Community package:
   ```bash
   npm install @astermind/astermind-community
   ```

3. Update imports:
   ```typescript
   // Old
   import { ELM } from '@astermind/astermind-elm';
   import { AdaptiveOnlineELM } from '@astermind/astermind-premium';
   import { OmegaSynth } from '@astermind/astermind-synthetic-data';
   
   // New (no license needed!)
   import { ELM, AdaptiveOnlineELM, OmegaSynth } from '@astermind/astermind-community';
   ```

4. Remove all license initialization code - it's no longer needed!
